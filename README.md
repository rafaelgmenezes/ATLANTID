# ATLANTID: a robust dolphin whistle classifier for the Southwestern Atlantic Ocean

ATLANTID is a whistle classifier designed for dolphin species in the Southwestern Atlantic. 
This repository contains the code and Jupyter Notebooks used for model development and validation.

## 🚀 Quick Start
This project uses **Jupyter Notebooks**. To run the classifier:
1. Clone this repository: `git clone https://github.com/rafaelgmenezes/ATLANTID.git`
2. Install the required libraries: `pip install -r requirements.txt`
3. Open `ATLANTID_classifier.ipynb` and follow the steps.

## 🛠 Software Requirements
* Python 3.x
* pandas, scikit-learn
* PAMGuard + ROCCA (for original feature extraction)
